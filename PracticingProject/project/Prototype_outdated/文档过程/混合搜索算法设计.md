## 混合搜索算法

系统采用 KNN 向量召回 + BM25 关键词重排序的两阶段混合检索策略。

**流程：** 查询文本经 Embedding 模型向量化后，KNN 召回 topK × 30 候选文档块，再以 BM25 关键词匹配（AND operator）对候选集重排序，最终返回 topK 结果。

**得分公式：**

```
final_score = 0.2 × KNN_similarity + 1.0 × BM25_score
```

- 向量相似度占 20% 权重作为语义 tie-breaker
- BM25 关键词匹配占主导，保证精确率
- 权限过滤（userId / isPublic / orgTag）同时作用于 KNN filter 和 query filter
- Embedding 调用失败时自动降级为纯文本检索
