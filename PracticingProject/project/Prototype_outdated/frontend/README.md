# Group45 - 企业智能 RAG 问答系统

本项目是 Group45 开发的企业智能 RAG 问答系统前端工程，用于对接企业知识库、检索增强生成服务和问答接口，为用户提供基于企业内部资料的智能问答体验。

## 项目说明

企业智能 RAG 问答系统面向企业知识管理场景，前端主要负责：

- 用户问答界面展示；
- 问题输入、提交和回答展示；
- 与后端 RAG 问答接口交互；
- 展示检索来源、引用内容或相关知识片段；
- 管理页面状态、加载状态、错误提示和基础交互体验。

RAG 是 Retrieval-Augmented Generation 的缩写，即检索增强生成。系统会先从企业知识库中检索与问题相关的内容，再结合大语言模型生成回答，从而提高回答的准确性和可追溯性。

## 环境要求

启动项目前，请确认本地已安装以下环境：

- Node.js：建议使用 LTS 版本；
- npm、pnpm 或 yarn：任选一种包管理器；
- Git：用于拉取和管理项目代码；
- 后端 RAG 服务：前端需要配置后端接口地址后才能完成完整问答流程。

如项目后续指定了精确版本，请以 `package.json`、锁文件或团队文档中的版本为准。
