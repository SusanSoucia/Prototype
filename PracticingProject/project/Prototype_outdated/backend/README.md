# Group45 企业知识问答后端

## 项目说明

本项目是小组内部协作开发的企业知识问答后端。系统面向企业内部知识库场景，负责文档解析、文本切分、向量化、检索和问答生成等后端能力。

项目目标：

- 支持企业知识文档的导入、管理和检索。
- 基于 RAG（Retrieval-Augmented Generation）流程生成回答。
- 降低大语言模型直接回答时的幻觉风险。
- 为企业内部知识查询、客服辅助、制度问答等场景提供服务端支撑。

## 技术栈

- Java 17
- Spring Boot 3.5.x
- Maven

## 环境要求

建议准备以下基础环境：

- Git
- JDK 17
- Maven 3.9 或以上版本
- 可用的大语言模型 API Key
- 可用的数据库或向量数据库服务

敏感配置不要直接写入代码仓库，应放在本地环境变量或 `.env` 文件中，并确保 `.env` 不提交到 Git。可以从 `.env.example` 复制一份作为本地启动配置。

## 本地运行

```bash
mvn spring-boot:run
```

## 测试

```bash
mvn test
```
