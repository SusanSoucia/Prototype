## 1. LiteParse

### 基本信息

| 项目 | 说明 |
|------|------|
| **类型** | 外部系统命令行工具（非 npm/Maven 依赖） |
| **命令名** | `lit`（可执行文件） |
| **在项目中的位置** | 无安装包，是服务器上的系统级命令行工具 |
| **获取地址** | https://github.com/opendatalab/LiteParse |
| **PyPI** | https://pypi.org/project/liteparse/ |

### 项目中的引用证据

**后端配置文件** `backend/src/main/resources/application.yml`:

```yaml
file:
  parsing:
    pdf:
      engine: ${FILE_PARSING_PDF_ENGINE:liteparse}
    liteparse:
      command: ${FILE_PARSING_LITEPARSE_COMMAND:lit}
      ocr-enabled: ${FILE_PARSING_LITEPARSE_OCR_ENABLED:true}
      ocr-language: ${FILE_PARSING_LITEPARSE_OCR_LANGUAGE:chi_sim+eng}
      tessdata-path: ${FILE_PARSING_LITEPARSE_TESSDATA_PATH:}
      max-pages: ${FILE_PARSING_LITEPARSE_MAX_PAGES:1000}
      dpi: ${FILE_PARSING_LITEPARSE_DPI:150}
      num-workers: ${FILE_PARSING_LITEPARSE_NUM_WORKERS:0}
      timeout-seconds: ${FILE_PARSING_LITEPARSE_TIMEOUT_SECONDS:300}
```

**后端 Java 服务类**:

| 文件 | 作用 |
|------|------|
| `backend/src/main/java/com/group45/backend/service/ParseService.java` | 核心 PDF 解析服务，包含 `parsePdfWithLiteParse()`、`buildLiteParseCommand()` 等方法 |
| `backend/src/main/java/com/group45/backend/service/LiteParseOcrAdapterService.java` | OCR 适配服务，接收 LiteParse 的 OCR 回调请求并转发到阿里云 OCR |
| `backend/src/main/java/com/group45/backend/controller/InternalOcrController.java` | REST 控制器，暴露 `POST /api/v1/internal/ocr/liteparse` 端点 |
| `backend/src/main/java/com/group45/backend/config/SecurityConfig.java` | 安全配置，对 `/api/v1/internal/ocr/**` 开放访问权限 |

### 安装方式

LiteParse 是 Python 生态的开源 PDF 解析工具，非 Maven 依赖，**需要在服务器上单独安装**：

```bash
# 安装 LiteParse（推荐 pip 安装）
pip install liteparse

# 确认命令可用
which lit
# 典型输出: /usr/local/bin/lit
```

生产部署配置（参考 `backend/docs/deployment.md` 第 4.5 节）：

```properties
FILE_PARSING_LITEPARSE_COMMAND=/usr/local/bin/lit
FILE_PARSING_LITEPARSE_OCR_ENABLED=false
FILE_PARSING_LITEPARSE_OCR_LANGUAGE=chi_sim
FILE_PARSING_LITEPARSE_DPI=300
```

> ⚠️ 注意：后台运行和 systemd 启动时 PATH 可能不同，建议使用绝对路径配置 `FILE_PARSING_LITEPARSE_COMMAND`。

### 项目中的引用

**文件** `frontend/src/components/custom/pdf-document-viewer.vue`:

```typescript
import { GlobalWorkerOptions, TextLayer, getDocument } from 'pdfjs-dist/legacy/build/pdf.min.mjs';
import type { PDFDocumentLoadingTask, PDFDocumentProxy, RenderTask } from 'pdfjs-dist';
```

## 4. UnoCSS

### 基本信息

| 项目 | 说明 |
|------|------|
| **包名** | `unocss` |
| **类型** | 前端 npm 依赖（开发依赖 + 自定义 preset） |
| **声明版本** | `^66.6.8`（`frontend/package.json` 第 94 行） |
| **实际安装版本** | `66.7.2` |
| **安装位置** | `frontend/node_modules/unocss/` |
| **获取地址** | https://www.npmjs.com/package/unocss |
| **官方文档** | https://unocss.dev |
| **GitHub** | https://github.com/unocss/unocss |

### 项目中的引用

**配置文件** `frontend/uno.config.ts`:

```typescript
import { defineConfig, transformerDirectives, transformerVariantGroup, presetWind3 } from 'unocss';
import { presetSoybeanAdmin } from '@sa/uno-preset';
import { themeVars } from './src/theme/vars';

export default defineConfig({
  content: {
    pipeline: {
      exclude: ['node_modules', 'dist']
    }
  },
  theme: {
    ...themeVars,
    fontSize: {
      'icon-xs': '0.875rem',
      'icon-small': '1rem',
      icon: '1.125rem',
      'icon-large': '1.5rem',
      'icon-xl': '2rem'
    }
  },
  shortcuts: {
    'card-wrapper': 'rd-8px shadow-sm'
  },
  transformers: [transformerDirectives(), transformerVariantGroup()],
  presets: [presetWind3({ dark: 'class' }), presetSoybeanAdmin()]
});
```

**Vite 插件注册** `frontend/build/plugins/unocss.ts`:
- 导入 `unocss/vite` 并配置 UnoCSS Vite 插件

**Vite 插件入口** `frontend/build/plugins/index.ts`:
- `setupUnocss(viteEnv)` — 注册到 Vite 插件管线

**自定义 Preset 包** `frontend/packages/uno-preset/src/index.ts`:
- 定义 `presetSoybeanAdmin`，包含 flex 布局、绝对/固定定位、文字溢出省略等快捷方式

**VS Code 配置**:
- `.vscode/extensions.json` — 推荐安装 `antfu.unocss` 扩展
- `frontend/.vscode/settings.json` — `"unocss.root": ["./"]`

### 安装方式

已声明在 `frontend/package.json` 的 `devDependencies` 中，通过 pnpm workspace 安装：

```bash
cd frontend
pnpm install
```
